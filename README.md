# Chapel-Hill-Crash-Data-Dashboard
Visualize Chapel Hill crash data.

Crash data from https://opendata-townofchapelhill.hub.arcgis.com/datasets/townofchapelhill::police-incident-reports-written/about

Traffic volumn data from https://connect.ncdot.gov/resources/State-Mapping/Pages/Traffic-Survey-GIS-Data.aspx

Using Leaflef for the base map, including different layer textures, a dotmap and a heatmap.
Using D3 for charts and graphs.
